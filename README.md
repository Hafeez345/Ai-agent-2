# 🤖 AI Data Analysis Agent (Web-Based)

This is a **web-based AI Data Analysis Agent** built using **Python and Streamlit**.  
It allows users to upload employee data in CSV format, perform advanced analysis, and ask interactive questions in **English or Urdu**.

---

## 🔹 Features

- **Upload CSV file** containing employee data (Name, ID, Salary, Rank, State)  
- **Analyze data** with pre-defined goals:
  1. Analyze Salary Statistics
  2. Analyze Rank Distribution
  3. Analyze State-wise Distribution
  4. Full Analysis Report
  5. Interactive Smart Q&A (ask any question about the data)
- **Interactive Q&A** using natural language (English/Urdu mix)  
- **Downloadable reports** in CSV or Excel format  
- **Portfolio-ready design** with clean interface
